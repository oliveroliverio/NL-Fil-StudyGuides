# How to Generate the Filipino Question Words Study Guide PDF

This guide explains how to use the provided Python script to generate a professional PDF study guide.

## Prerequisites

You need to have **Python 3** installed on your computer.

## Step 1: Install Dependencies

The script uses the `reportlab` library to create the PDF. You need to install it using `pip`.

Open your terminal or command prompt and run:

```bash
pip install reportlab
```

*Note: If `pip` is not recognized, try `pip3` or `python -m pip`.*

## Step 2: Run the Script

1.  Ensure you have the file `generate_filipino_pdf.py` in your current directory.
2.  Run the script by typing:

```bash
python generate_filipino_pdf.py
```

*Or if you use Python 3 explicitly:*

```bash
python3 generate_filipino_pdf.py
```

## Step 3: Check the Output

After running the script, you should see a message saying:
`PDF generated successfully: filipino_question_words_study_guide.pdf`

Check your folder for the new PDF file named `filipino_question_words_study_guide.pdf`.
