# Basic Time Adverbs

| Filipino    | English               | Example Sentence         | English Translation       |
| ----------- | --------------------- | ------------------------ | ------------------------- |
| **ngayon**  | now                   | Ngayon ako aalis.        | I’m leaving now.          |
| **mamaya**  | later                 | Mamaya tayo kumain.      | Let’s eat later.          |
| **kanina**  | earlier / a while ago | Kanina siya umalis.      | He left a while ago.      |
| **kahapon** | yesterday             | Kahapon kami naglakad.   | We walked yesterday.      |
| **bukas**   | tomorrow              | Bukas ang exam ko.       | My exam is tomorrow.      |
| **noon**    | long ago              | Noon siya nakatira dito. | He lived here long ago.   |
| **kagabi**  | last night            | Kagabi ako nag-aral.     | I studied last night.     |
| **minsan**  | sometimes             | Minsan ako nalilito.     | Sometimes I get confused. |
| **lagi**    | always                | Lagi akong pagod.        | I’m always tired.         |
| **madalas** | often                 | Madalas siyang huli.     | He’s often late.          |
| **bihira**  | rarely                | Bihira siyang lumabas.   | He rarely goes out.       |
