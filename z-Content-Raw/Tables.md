# Basic Time Adverbs
| Filipino    | English               | Example Sentence         | English Translation       |
| ----------- | --------------------- | ------------------------ | ------------------------- |
| **ngayon**  | now                   | Ngayon ako aalis.        | I’m leaving now.          |
| **mamaya**  | later                 | Mamaya tayo kumain.      | Let’s eat later.          |
| **kanina**  | earlier / a while ago | Kanina siya umalis.      | He left a while ago.      |
| **kahapon** | yesterday             | Kahapon kami naglakad.   | We walked yesterday.      |
| **bukas**   | tomorrow              | Bukas ang exam ko.       | My exam is tomorrow.      |
| **noon**    | long ago              | Noon siya nakatira dito. | He lived here long ago.   |
| **kagabi**  | last night            | Kagabi ako nag-aral.     | I studied last night.     |
| **minsan**  | sometimes             | Minsan ako nalilito.     | Sometimes I get confused. |
| **lagi**    | always                | Lagi akong pagod.        | I’m always tired.         |
| **madalas** | often                 | Madalas siyang huli.     | He’s often late.          |
| **bihira**  | rarely                | Bihira siyang lumabas.   | He rarely goes out.       |

# Common time phrases

| Filipino Phrase      | English                | Example Sentence                   | English Translation                |
| -------------------- | ---------------------- | ---------------------------------- | ---------------------------------- |
| **kaninang umaga**   | earlier this morning   | Kaninang umaga ako nag-jogging.    | I jogged earlier this morning.     |
| **kaninang hapon**   | earlier this afternoon | Kaninang hapon ka tumawag.         | You called earlier this afternoon. |
| **kaninang gabi**    | earlier tonight        | Kaninang gabi siya dumating.       | He arrived earlier tonight.        |
| **ngayong araw**     | today                  | Ngayong araw ang biyahe.           | The trip is today.                 |
| **isang araw**       | one day / someday      | Isang araw, bibisita ako sa Japan. | One day, I’ll visit Japan.         |
| **noong nakaraan**   | previously             | Noong nakaraan ko siya nakita.     | I saw him previously.              |
| **sa madaling-araw** | at dawn                | Sa madaling-araw kami umuwi.       | We went home at dawn.              |
| **sa huli**          | in the end             | Sa huli, pumayag siya.             | In the end, he agreed.             |
| **sa ngayon**        | for now                | Sa ngayon, pahinga muna.           | For now, rest first.               |

# Duration words

| Filipino                 | English               | Example                         | English Translation         |
| ------------------------ | --------------------- | ------------------------------- | --------------------------- |
| **matagal**              | long time             | Matagal na kitang kilala.       | I’ve known you a long time. |
| **sandali / saglit**     | a moment              | Saglit lang ako.                | I’ll just be a moment.      |
| **maikli / saglit lang** | short time            | Maikli lang ang meeting.        | The meeting is short.       |
| **madaling-araw**        | dawn (period of time) | Madaling-araw na kami nagising. | We woke up at dawn.         |
| **maghapon**             | the whole day         | Nagtrabaho siya maghapon.       | He worked all day.          |
| **magdamag**             | all night             | Nag-aral ako magdamag.          | I studied all night.        |

# Aspect like particles

| Particle    | Meaning                 | Function                        | Example             | English Translation     |
| ----------- | ----------------------- | ------------------------------- | ------------------- | ----------------------- |
| **na**      | already                 | marks completion or “up to now” | Nandito na ako.     | I’m already here.       |
| **pa**      | still / yet             | continues something             | Nandito pa ako.     | I’m still here.         |
| **pala**    | oh / actually           | new realization                 | Bago ka pala!       | Oh, you’re new!         |
| **muna**    | for now                 | temporary pause                 | Dito ka muna.       | Stay here for now.      |
| **lang**    | just / only             | limits extent or time           | Bago ka lang.       | You’re just new.        |
| **naman**   | meanwhile / in contrast | softens tone                    | Sandali lang naman. | It’ll only be a moment. |
| **na lang** | just / instead          | alternative or resignation      | Bukas na lang.      | Just do it tomorrow.    |
